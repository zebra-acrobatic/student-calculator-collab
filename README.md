# 🧮 Student Calculator Collaboration Project

Welcome to the collaborative calculator development project! This repository is designed for students to practice professional Git workflows, issue tracking, and collaborative development.

## 🎯 Project Overview

This is a simple Python calculator that students will enhance through collaborative development. Each student should:
1. Create GitHub Issues for new features
2. Implement features in separate branches
3. Submit Pull Requests for code review
4. Practice resolving merge conflicts

## 🚀 Getting Started

### Prerequisites
- VS Code installed
- Git configured with your GitHub account
- Basic Python knowledge

### How to Contribute

1. **Explore existing issues** to see what's planned
2. **Create a new issue** using our templates for your proposed feature
3. **Wait for approval** from instructors before starting work
4. **Clone the repository** to your local machine
5. **Create a feature branch** following our naming conventions
6. **Implement your feature** following our coding standards
7. **Submit a Pull Request** with comprehensive documentation

## 🔧 Development Guidelines

### Branch Naming Convention
- `feature/description-issue-number` (e.g., `feature/add-division-issue-23`)
- `bugfix/description-issue-number` (e.g., `bugfix/fix-negative-input-issue-45`)

### Commit Message Format
- Use clear, descriptive commit messages
- Reference issue numbers: `"Add division functionality (#23)"`

## 🤝 Code of Conduct

- Be respectful and constructive in all interactions
- Follow the coding standards outlined in CONTRIBUTING.md
- Test your code thoroughly before submitting PRs
- Write clear documentation for new features

## 📞 Getting Help

- Check existing issues and discussions
- Ask questions in PR comments
